# 작업 지침

`src/main/kotlin/com/sight` 아래의 코드를 새로 만들거나 수정하기 전에, 수정할 파일이 속한 레이어의 `AGENTS.md`를 처음부터 끝까지 읽고 따른다.

- `config`의 파일을 수정할 때는 `src/main/kotlin/com/sight/config/AGENTS.md`를 읽는다.
- `controllers`의 파일을 수정할 때는 `src/main/kotlin/com/sight/controllers/AGENTS.md`를 읽는다.
- `core`의 파일을 수정할 때는 `src/main/kotlin/com/sight/core/AGENTS.md`를 읽는다.
- `domain`의 파일을 수정할 때는 `src/main/kotlin/com/sight/domain/AGENTS.md`를 읽는다.
- `repository`의 파일을 수정할 때는 `src/main/kotlin/com/sight/repository/AGENTS.md`를 읽는다.
- `service`의 파일을 수정할 때는 `src/main/kotlin/com/sight/service/AGENTS.md`를 읽는다.

두 개 이상 레이어의 파일을 수정할 때는 관련된 모든 `AGENTS.md`를 읽는다. 두 규칙이 서로 다르면, 더 넓은 동작을 금지하는 규칙을 따른다.
