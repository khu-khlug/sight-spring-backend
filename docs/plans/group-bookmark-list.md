## 북마크 목록 조회 API

### 비즈니스 규칙

- 본인의 북마크 목록만 조회 가능

### 관련 DB 테이블

| 테이블           | 역할                        |
| ---------------- | --------------------------- |
| `group_bookmark` | 즐겨찾기 row 조회           |

### API

```
GET /users/:userId/bookmark/
```

#### 응답

```
200 OK
```

| 이름                | 타입      | 설명               |
| ------------------- | --------- | ------------------ |
| `bookmarks`         | 객체 배열 | 즐겨찾기 그룹 목록 |
| `bookmarks.groupId` | 정수      | 그룹 ID            |

#### 테스트 케이스

1. 본인이 조회 요청 → 200
2. 다른 회원의 북마크 조회 요청 → 403
3. 존재하지 않는 `userId` 요청 → 404
