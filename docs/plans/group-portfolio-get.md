## 포트폴리오 조회 API

### 비즈니스 규칙

- 누구나 조회 가능

### 관련 DB 테이블

| 테이블  | 역할                        |
| ------- | --------------------------- |
| `group` | `portfolio` 컬럼 조회       |

### API

```
GET /groups/:groupId/portfolio
```

#### 응답

```
200 OK
```

| 이름        | 타입    | 설명           |
| ----------- | ------- | -------------- |
| `published` | boolean | 현재 발행 여부 |

#### 테스트 케이스

1. 조회 요청 → 200
2. 존재하지 않는 그룹 ID 요청 → 404
